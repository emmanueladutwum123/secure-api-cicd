# Security Policy

## Reporting a Vulnerability

Please email **emmanuelsoneuclid10@gmail.com** with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

Do not open a public GitHub issue for security vulnerabilities.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.1.x   | Yes       |
| < 1.1   | No        |
